Q2. LocalStorage vs SessionStorage

LocalStorage:Local storage is not unique per tab. Local storage store data without expire date, although we close our browser and then open this we mus see the previous data.At safari Local storage has limited storage 2551k characters. SessionStorage:Session storage is unique per tab. Session storage also store data but When we close our browser or tab this data is going to deleted. LocalStorage and SessionStorage both data can get and also set . At safari SessionStorage has unlimited storage. No difference in speed. On google chrome browser both hav limited storage 5120k characters.
